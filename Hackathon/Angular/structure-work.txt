После создания проекта структура файлов будет примерно такой:

my-angular-app/  
├── src/  
│   ├── app/  <-- Здесь основной код приложения  
│   │   ├── app.component.ts  <-- Логика главного компонента  
│   │   ├── app.component.html  <-- HTML главного компонента  
│   │   ├── app.component.css  <-- Стили главного компонента  
│   │   ├── app.module.ts  <-- Модуль, в котором регистрируются компоненты  
│   ├── assets/  <-- Статические файлы (изображения, шрифты)  
│   ├── index.html  <-- Главный HTML-файл приложения  
│   ├── styles.css  <-- Глобальные стили  
│   ├── main.ts  <-- Точка входа в приложение  



Где писать код для страницы?
Основная работа ведётся в src/app/.

1. Редактирование главной страницы
Файл: src/app/app.component.html
Здесь можно изменить содержимое страницы. Например:

<h1>Привет, Angular!</h1>
<p>Это моя первая страница.</p>



Файл: src/app/app.component.css (для стилей)

h1 {
  color: blue;
  text-align: center;
}





2. Добавление нового компонента (например, "О нас")

1. Создайте новый компонент с помощью команды:

ng generate component about

(Создаст папку src/app/about/ с файлами компонента)


2. Откройте src/app/about/about.component.html и напишите код:

<h2>О нас</h2>
<p>Мы создаем веб-приложения на Angular!</p>


3. В src/app/app.component.html добавьте:

<app-about></app-about>

4. Теперь этот компонент будет отображаться на странице.
