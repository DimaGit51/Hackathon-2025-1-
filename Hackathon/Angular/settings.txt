Для создания веб-приложения на Angular нужно выполнить несколько простых шагов:

1. Убедитесь, что у вас установлен Node.js и npm. Вы можете проверить их версии с помощью команд:
node -v
npm -v

2. Установите Angular CLI, если он еще не установлен:
npm install -g @angular/cli

3. Создайте новый проект Angular:
ng new my-angular-app

Здесь my-angular-app — это название вашего проекта. Во время создания вам предложат выбрать некоторые параметры (например, хотите ли вы использовать маршрутизацию или какой стиль CSS предпочитаете).

4. Перейдите в папку проекта:
cd my-angular-app

5. Запустите приложение:
ng serve

Это запустит сервер разработки, и ваше приложение будет доступно по адресу: http://localhost:4200
